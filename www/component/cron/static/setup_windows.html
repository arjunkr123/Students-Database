To configure scheduled tasks in Windows, you need to go to Control Panel > Administrative Tools > Task Scheduler.
To execute the scheduled tasks of Students Management Software, you need to execute 2 PHP files
  • cron.php every few minutes (at least every 5 minutes would be the best, every 2 minutes recommended)
  • cron_maintenance.php every night (when no user will use the application)
To execute a PHP file, in the Task Scheduler, click on Create Task, then
  • Configure the task to be run by the same user as Apache (usually, if the Apache server is running as a service, it will use the SYSTEM user)
  • Configure also that the task will be executed whether the user is logged on or not
  • In the triggers, configure when the task will be executed (every n minutes for the first one, daily for the second one)
  • In the actions, create an action Start a program:
    • launch the executable php.exe located where PHP is installed (example: C:\PHP5\php.exe)
    • configure it to start in the directory where Students Management Software is installed (example: C:\StudentsManagement)
    • as argument, specify the option -f followed by the full path of the php file to execute (cron.php for the first task, cron_maintenance.php for the second task). example: -f "C:\StudentsManagement\cron.php"
