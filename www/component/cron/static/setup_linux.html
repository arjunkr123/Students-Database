To configure scheduled tasks in Linux, you need to add entries in your /etc/crontab file.
To execute tasks, you need to execute 2 PHP files
  • cron.php every few minutes (at least every 5 minutes would be the best, every 2 minutes recommended)
  • cron_maintenance.php every night (when no user will use the application)
Here is an exemple of the 2 entries to add in /etc/crontab:
*/2 * * * * wwwrun cd /srv/www/vhosts/students-management/ && /usr/bin/php cron.php
0 1 * * * wwwrun cd /srv/www/vhosts/students-management/ && /usr/bin/php cron_maintenance.php

Where /srv/www/vhosts/students-management/ is the directory where Students Management Software is installed.
The first line, executes every 2 minutes the file cron.php, using the PHP command line executable (here /usr/bin/php), and using user wwwrun which is the Apache user
The second line is similar, to execute every day at 1AM the file cron_maintenance.php
